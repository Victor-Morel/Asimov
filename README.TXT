Projet réalisé par :
Victor Morel, Logan Paul, Théo Dury

Ce projet a été conçu avec Java7 et l’IDE Intellij IDEA 14.1.3, et les tests avec JUnit 4.1.12

Nous avons utilisé GitHub comme forge logicielle.
URL de clonage : git@github.com:Victor-Morel/Asimov.git

Pour exécuter le programme, lancer Asimov.jar.

Pour compiler le programme, pointer sur run/main.java.

Pour ajouter la javadoc, charger le fichier javadoc.jar.

Pour ajouter les libs ajouter les fichiers junit-4.12 et hamcrest-all-1.3.

Pour lancer les tests, exécuter les fichiers de tests inclus dans les packages de test.model :
Se placer sur le fichier de tests, clic droit “Run testMonFichier”.
